# Booru Userscripts

A collection of userscripts designed to enhance your experience on various booru image boards.

## Scripts in this folder

### [Booru Search Extended](booru-search-extended.md)
**File:** `booru-search-extended.user.js`

A powerful userscript that enhances tag searching on multiple booru image boards with an advanced tag builder interface.

**Key Features:**
- Visual tag builder with tree-based UI
- Toggle tag visibility
- Multi-site support (Rule34, Gelbooru, Danbooru, and more)
- Favorites system with import/export
- Drag & drop reordering
- Query parsing

### [Booru Downloader](booru-downloader.md)
**File:** `booru-downloader.js`

Press Ctrl+S on an image page to save it with configurable download modes.

**Key Features:**
- Quick download with Ctrl+S
- Configurable filename formats
- Multiple download modes
- Site metadata integration

### [Booru Tag Blocker](tag-blocker.md)
**File:** `tag-blocker.user.js`

Block booru tags on the front-end with persistence and UI.

**Key Features:**
- Front-end tag filtering
- Customizable blocklist
- Hide or blur blocked content
- Persistent settings

## Installation

All scripts require a userscript manager:
- [Tampermonkey](https://www.tampermonkey.net/) (Chrome, Firefox, Safari, Edge)
- [Violentmonkey](https://violentmonkey.github.io/) (Chrome, Firefox, Edge)
- [Greasemonkey](https://www.greasespot.net/) (Firefox)

Click the individual script documentation links above for specific installation instructions.
