# Stash Userscripts

A collection of userscripts to enhance [Stash](https://github.com/stashapp/stash) - a self-hosted web-based pornography organizer.

## Scripts in this folder

### [Stash Gallery URL Opener](stash-gallery-url-opener.md)
**File:** `stash-gallery-url-opener.user.js`

Add open buttons to gallery URLs in Stash for quick access to external gallery sources.

**Key Features:**
- One-click opening of gallery URLs
- Convenient buttons next to URL fields
- Works in gallery edit mode

## Installation

All scripts require a userscript manager:
- [Tampermonkey](https://www.tampermonkey.net/) (Chrome, Firefox, Safari, Edge)
- [Violentmonkey](https://violentmonkey.github.io/) (Chrome, Firefox, Edge)
- [Greasemonkey](https://www.greasespot.net/) (Firefox)

Click the individual script documentation links above for specific installation instructions.

## About Stash

[Stash](https://github.com/stashapp/stash) is a self-hosted pornography organizer with tag-based categorization, scene and performer management, and a powerful GraphQL API. These scripts extend its functionality by automating metadata import from popular gallery sites.
